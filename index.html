How to setup a Thingspeak Account.

Step 1: Your Digital Passport

To use ThingSpeak, you first have to join the service by creating a free account.

  • Visit ThingSpeak.com and click "Sign Up".
  • Follow the prompts to create a MathWorks Account.
  • Crucial: Check your inbox for a verification email. You must click the link inside to activate your account.
2

Step 2: Create a Data "Vault"

Next, you must create a Channel in the service to receive our data from the sensor.

New Channel Settings
Name: Library Monitor
Field 1: Temperature
Field 2: Humidity
  1. Click Channels > My Channels > New Channel.
  2. Give it a Name and tick the checkboxes for Field 1 and Field 2.
  3. Scroll to the bottom and click Save Channel.
3

Step 3: The Secret Key (API Key)

Now you will use a special key that will be assigned to our project by ThingSpeak, called the API Key.

ThingSpeak Dashboard Example:
Channel Settings API Keys Data Import
Write API Key
5Q0JY180IP1076ES
API Requests (How the computer sends data)
GET https://api.thingspeak.com/update?api_key=5Q0JY180IP1076ES&field1=0
How to use it in your Code:

You will paste your assigned key into your wificonfig.py file like this:

THINGSPEAK_WRITE_API_KEY = '5Q0JY180IP1076ES'
4

Step 4: Pin It on the Map

Make known the location of our sensor

Finally, we need to tell ThingSpeak where the sensor is physically located.

  1. Open Google Maps and search for "Tampines Regional Library".
  2. Right-Click on the building on the map.
  3. The coordinates appear at the very top of the pop-up menu!
  4. Click those numbers to copy them automatically.
