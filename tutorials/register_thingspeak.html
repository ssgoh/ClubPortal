Cloud Registration

To visualise sensor data online, we use ThingSpeak. This guide walks you through the account setup process.

1. Join Service
2. Create Channel
3. Get API Key
4. Map Location
1

Create Your Account

Visit ThingSpeak.com and click "Sign Up". You must verify your email to continue.

2

The Data "Vault" (Channel)

Navigate to Channels > My Channels > New Channel. Set the name to "Library Monitor" and enable Field 1 for Temperature.

3

Your Secret Write API Key

Find your key under the API Keys tab. Copy this into wificonfig.py.
