TM1637 Display Guide

Master the 4-digit 7-segment display with MicroPython and Raspberry Pi Pico W.

1. Download Library

You need the tm1637.py library to interface with the chip.

  1. Open Thonny IDE with your Pico W connected.
  2. Go to Tools > Manage Packages.
  3. Search for micropython-tm1637 and install it.
  4. Alternatively, create a new file named tm1637.py on your Pico and paste the library code manually.

Displaying Words

You can show basic words (limited by 7-segment logic) or set brightness levels.

import tm1637
from machine import Pin

# Setup Display (CLK=16, DIO=17)
display = tm1637.TM1637(clk=Pin(16), dio=Pin(17))

# Set Brightness (0-7)
display.brightness(3)

# Show "COOL"
display.show("COOL")

Displaying Numbers

Perfect for sensor readings, counters, or temperature displays.

# Show an integer
display.number(1234)

# Show temperature with 'C' suffix
display.show("24 C")

# Clear the display
display.fill(0)

Time and Colon Control

Control the center colon using the `colon` parameter or by toggling individual segments.

# Simple Clock Display
hour = 12
minute = 30

# Display with colon ON
display.numbers(hour, minute, colon=True)

# Logic for a blinking colon
import time
while True:
    # Colon ON
    display.numbers(12, 00, True)
    time.sleep_ms(500)
    # Colon OFF
    display.numbers(12, 00, False)
    time.sleep_ms(500)
