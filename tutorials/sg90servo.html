SG90 Servo Tutorial

Micro-Actuator Control with Raspberry Pi Pico W

Servo Fundamentals

The SG90 is an analog servo that moves its output shaft to a specific angle based on a PWM signal. Unlike a DC motor, it maintains its position as long as the signal is provided.

Pin Connections

  • Orange: GP15 (Signal)
  • Red: VBUS (5V Power)
  • Brown: GND (Ground)
SIMULATION: 0°

SG90 Servo Wiring 

from machine import Pin, PWM
import utime

# Setup PWM on GP15
servo = PWM(Pin(15))
servo.freq(50) # 50Hz = 20ms period

def set_servo_angle(angle):
    # duty = percentage of arc * range + offset
    duty = int(((angle / 180) * 6554) + 1638)
    servo.duty_u16(duty)

while True:
    for target in [0, 45, 90, 135, 180]:
        print(f"Moving to: {target}°")
        set_servo_angle(target)
        utime.sleep(1)

The Math Behind the Signal

Why 65,535?

The Pico uses 16-bit PWM resolution. A 16-bit number ranges from 0 to 65,535 ($2^{16} - 1$).

Step-by-Step Derivation

1. Finding the Period (The Window)

1 second / 50 Hz = 20ms total time

2. Calculating the "Granularity"

20ms / 65,535 ≈ 0.000305 ms per step

3. Finding Pulse Values

0° Position (0.5ms)
0.5 / 0.000305 ≈ 1638
180° Position (2.5ms)
2.5 / 0.000305 ≈ 8192

The Duty Cycle Equation

duty = int(((angle / 180) * 6554) + 1638)

(angle / 180)

Percentage of total travel (0.0 to 1.0).

* 6554

Scaled to the movement range (8192 - 1638).

+ 1638

Adds the starting offset so 0° begins at the required 0.5ms pulse mark.
