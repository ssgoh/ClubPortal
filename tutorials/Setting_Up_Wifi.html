Variable Abstraction

                            
Instead of hardcoding details inside functions, variables create a "Configuration Zone". This makes code portable across different networks without risking the logic.

                        

                        

                            
The "Status 3" Rule

                            
MicroPython status codes are internal markers. 3 specifically indicates that IP assignment (DHCP) was successful and the hardware is fully "Station Active".

2.4GHz Only

                        
The Pico W does not support 5GHz bands. Check your router settings for a legacy 2.4GHz broadcast.

                    

                    

                        
Firmware Check

                        
If import network fails, ensure you flashed the Pico W specific MicroPython UF2 file.

                    

                    

                        
Signal Block

                        
Metal enclosures or placing fingers on the silver antenna will drop signal strength significantly.
